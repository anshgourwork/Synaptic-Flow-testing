# Synaptic Flow | AI-Powered Business Automation

**Synaptic Flow** is a high-performance, premium website for an AI automation agency. It features 3D interactive elements, a fluid modern layout, and is optimized for speed and conversion.

## 🚀 Deployment
Live on Vercel: [https://synaptic-flow-steel.vercel.app](https://synaptic-flow-steel.vercel.app)

## 🛠 Features
- **3D Spline Integration**: Interactive robot and background abstract elements.
- **Glassmorphic UI**: Minimal, modern design with frosted glass effects.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **SEO Optimized**: Pre-configured meta tags, OG tags, and structured heading hierarchy.
- **Fast Delivery**: Preloaded assets and minimal dependencies.

## 📂 Project Structure
- `index.html`: Main application entry point.
- `style.css`: Core design system and layout.
- `script.js`: Interactive logic and animations.
- `LOGO main.png`: Brand asset.
- `neuehaasgrotdisp-55roman-trial.otf`: Custom corporate typography.
